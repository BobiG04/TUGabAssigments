# 🌐 Social Network Project

Този проект е разработен като **Курсов проект** за Семестър 6 в ТУ-Габрово. Представлява уеб-базирана социална мрежа, фокусирана върху взаимодействието между потребители, споделянето на съдържание и управлението на профили.

## 🚀 Технологичен стек

Проектът е изграден с модерни технологии за осигуряване на бързина и скалируемост:

*   **Frontend:** React.js, Tailwind CSS, Vite
*   **Backend:** Node.js / Express (или съответната технология, използвана в проекта)
*   **База данни:** MongoDB / PostgreSQL
*   **State Management:** Context API / Redux

## ✨ Основни функционалности

*   **Потребителска аутентикация:** Регистрация, вход и управление на сесии.
*   **Профили:** Персонализиране на потребителска информация и снимки.
*   **Публикации:** Създаване, редактиране и изтриване на постове.
*   **Взаимодействие:** Система за харесвания (likes) и коментари.
*   **Приятелства:** Възможност за добавяне и премахване на приятели/последватели.

## 🛠️ Инсталация и стартиране

1. Клонирайте хранилището:
   ```bash
   git clone https://github.com/your-username/TUGabAssigments.git
   ```
2. Навигирайте до директорията на проекта:
   ```bash
   cd Year_3/Semester_6/Course_Project/social-network-project
   ```
3. Инсталирайте зависимостите:
   ```bash
   npm install
   ```
4. Стартирайте приложението в режим на разработка:
   ```bash
   npm run dev
   ```
