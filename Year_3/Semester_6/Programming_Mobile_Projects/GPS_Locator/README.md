# GPS Locator

A mobile application designed to track and display real-time GPS coordinates.

## Getting Started

Follow these instructions to get the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Flutter SDK or Node.js (depending on the branch/version)
- Android Studio / Xcode for mobile emulation

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TUGabAssigments/tree/master/Year_3/Semester_6/Programming_Mobile_Projects/GPS_Locator.git
   ```
2. Install dependencies:
   ```bash
   flutter pub get # or npm install
   ```
3. Run the application:
   ```bash
   flutter run # or npm start

   ```
