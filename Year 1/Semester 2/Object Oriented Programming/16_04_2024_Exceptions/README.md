# 🗓️ Упражнение: 16.04.2024 - Обработка на изключения (Exceptions)

Тази директория съдържа решения на задачите от Семинарно упражнение № 4. Фокусът е върху предпазването на програмите от неочаквани грешки (сривове) по време на изпълнение чрез използването на конструкциите `try`, `catch`, `finally` и ръчно предизвикване на грешки чрез `throw`.

## 🗂️ Списък със задачите

| Проект       | Описание по условие                                                                                                                                                                                                                                   |
| :----------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. _1zad** | Деление на две числа с прихващане на грешки (напр. деление на нула). Демонстрира блок `finally` за гарантирано изпълнение на код накрая.                       |
| **2. _2zad** | Пресмятане на корен квадратен. Ръчно генериране на изключение (`throw new Exception()`), ако въведеното число е по-малко или равно на нула.                             |
| **3. _3zad** | Метод, връщащ име на месец по даден номер. Съдържа защита срещу въвеждане на невалиден номер на месец чрез изключения.                                                |
| **4. _4zad** | Клас `Time`, валидиращ въведените стойности за час (0-23), минута (0-59) и секунда (0-59). При невалидни данни програмата генерира и обработва изключение. |
