## 🚀 Как се стартират проектите
Всички проекти са стандартни C# конзолни приложения. За да стартирате даден проект:
1. Отворете съответната папка (напр. `PineTree`).
2. Отворете `.sln` или `.csproj` файла във Visual Studio.
3. Стартирайте с `F5` или `Ctrl+F5`.
