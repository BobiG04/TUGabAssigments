# 🗓️ Упражнение: 02.04.2024 - Свойства на класовете (Properties)

Тази директория съдържа решения на задачите от Лабораторно упражнение № 8. Проектите са фокусирани върху концепцията за капсулация в ООП чрез използване на свойства (`Properties` с `get` и `set` аксесоари), включително свойства само за четене (read-only) и само за запис (write-only).

## 🗂️ Списък със задачите

| Проект       | Описание по условие                                                                                                                                                                                                                                                                                                                                   |
| :----------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. _1zad** | Клас `Rectangle`. Демонстрира свойства **само за запис** (write-only) за ширина и височина, и свойство **само за четене** (read-only) за пресмятане на площта. Програмата намира правоъгълника с най-голяма площ. |
| **2. _2zad** | Клас `CarVroom` (Автомобил). Демонстрира пълни свойства (read & write) за марка, цвят и регистрационен номер.                                                                                                                                                                                    |
| **3. _3zad** | Клас `Person` (Персона). Съхранява данни за име, фамилия и телефонен номер с пълни свойства и конструктори.                                                                                                                                                                           |
