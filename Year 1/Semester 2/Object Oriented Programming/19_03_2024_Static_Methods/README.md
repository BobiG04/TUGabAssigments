# 🗓️ Упражнение: 19.03.2024 - Статични методи

Тази директория съдържа задачи от Лабораторно упражнение № 5. Фокусът е върху създаването и използването на статични методи (`static`) и разбирането на разликата между метод на класа и метод на екземпляра (обекта).

## 🗂️ Списък със задачите

| Проект | Описание по условие | Статус |
| :--- | :--- | :--- |
| **1. _1zad** | Клас `Dot`, описващ точка в равнината. Демонстрира изчисляване на разстояние между две точки чрез статичен метод, приемащ два обекта от тип `Dot` като параметри. | Завършена |
| **2. _2zad** | Клас `Converter`, съдържащ статични методи за преобразуване на скорост: от метри в секунда (m/s) в километри в час (km/h) и обратно. | Завършена |
| **3. _3zad** | Клас `Complex`, описващ комплексно число (реална и имагинерна част). Включва статичен метод за събиране на две комплексни числа. | Завършена |
