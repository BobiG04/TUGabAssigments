# 🗓️ Упражнение: 14.05.2024 - Абстрактни класове и Интерфейси

Тази директория съдържа решения на задачите от Лабораторно упражнение № 11 (Интерфейси и Абстрактни класове). Фокусът е върху постигането на пълна абстракция, принудителен полиморфизъм и дефиниране на договори (contracts) чрез интерфейси.

## 🗂️ Списък със задачите

| Проект           | Описание по условие                                                                                                                                                                                                                                                                                                             | Статус         |
| :--------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------- |
| **1. Figures**   | Абстрактен базов клас `Figure` с абстрактни методи. Производни класове `Square`, `Circle`, `Rhombus`, `Rectangle`. Демонстрира замяна на метод със свойство за четене и имплементация на интерфейс `IName`. | Завършена   |
| **2. Predators** | Интерфейс `IPreditor`, описващ поведение на животно (ходене, бягане, хранене, звук). Имплементиращи класове `Tiger`, `Lion`, `Cat`.                                                                                                                        | *Предстои* |
