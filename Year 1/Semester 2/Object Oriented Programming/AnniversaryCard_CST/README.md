# 🎈 Поздравителна картичка: 40 години КСТ

Тази директория съдържа креативно конзолно приложение, създадено по случай 40-годишнината на катедра "Компютърни системи и технологии" (КСТ) в Технически университет – Габрово.

## ⚙️ Описание на проекта

Програмата изчертава поздравителна картичка директно в конзолата, използвайки ASCII символи за рамка (`┌`, `─`, `┐`, `│`, `└`, `┘`) и прецизно изчислени интервали за перфектно центриране на текста.

* **Технология:** C# Console Application
* **Функционалност:** Алгоритмично чертане на рамка чрез `for` цикъл и условни оператори (`if-else`) за вмъкване на поздравителните съобщения на точните редове.
