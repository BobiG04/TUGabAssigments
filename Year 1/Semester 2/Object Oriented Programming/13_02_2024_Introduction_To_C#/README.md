# 🗓️ Упражнение: 13.02.2024 - Първи стъпки в C#

Тази директория съдържа първите проекти на C#, фокусирани върху базов синтаксис, работа с типове данни, създаване на методи и условни конструкции.

## 🗂️ Списък със задачите

| Проект (Папка) | Описание на задачата                                                                                                                                                                                                                                                                         |
| :------------------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. Tickets**      | Пресмятане на бюджет за концерт на Бионсе. Изчислява транспортни разходи според броя на хората и проверява дали остатъкът от бюджета стига за VIP или нормални билети. |
| **2. StValentine**  | Калкулатор за букети за Свети Валентин. Пресмята цени на различни цветя (рози, лалета, орхидеи) с включени празнични надценки и специфични сезонни отстъпки.                  |
| **3. Area**         | Използване на методи за пресмятане на лица на геометрични фигури (кръг, успоредник, квадрат, триъгълник, трапец).                                                                                                  |
| **4. Formulas**     | Пресмятане на сложни математически изрази чрез използване на вградения клас `Math` в C# (`Math.Pow`, `Math.Sqrt`, `Math.Abs`).                                                                                                      |
| **5. VarSizes**     | Демонстрационна програма, която извежда размера в байтове на различните примитивни типове данни в C# чрез оператора `sizeof()`.                                                                             |

## 🧮 Математически формули (Проект 4)

В проект `Formulas` са реализирани следните математически изрази:

1. $y = \frac{1}{\frac{1}{x-1}}$
2. $y = \sqrt{(x^2 - 5)(x + 3)}$
3. $y = \frac{(a+b)^2 - 4a^3}{\sqrt{3|a|} + 1}$
4. $y = 1 - \frac{(x^2-1)^5}{\sqrt{x^2+2x+1}}$
5. $y = \frac{5-x^3}{3} - \frac{x-1}{x+3}$
6. $y = \left| \frac{\sqrt{2+x^5}}{x^4+5x} \right|$
