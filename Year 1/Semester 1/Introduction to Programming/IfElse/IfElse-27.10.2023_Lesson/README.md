# 🔀 Разширени условни оператори и Switch-Case

Тази директория съдържа задачи, фокусирани върху по-сложни логически проверки, вложени условни оператори и изграждане на базови менюта чрез `switch-case`.

## 🗂️ Списък със задачите

| Задача | Описание на логиката |
| :--- | :--- |
| **Задача 1** | Сравняване на две цели числа и извеждане на по-голямото чрез прост `if-else`. |
| **Задача 2** | Определяне на вида на триъгълник (равностранен, разностранен, равнобедрен) на базата на три въведени дължини на страни. |
| **Задача 3** | Проверка за валидност на триъгълник (чрез неравенството на триъгълника). При валидност се определя вида му и се пресмята лицето му по Хероновата формула. |
| **Задача 4** | Решаване на квадратно уравнение ($ax^2 + bx + c = 0$). Изчислява дискриминантата ($D = b^2 - 4ac$) и намира реалните корени, ако съществуват. |
| **Задача 5** | Пресмятане на стойността на стъпаловидна (piecewise) функция в зависимост от интервала, в който попада променливата $x$. |
| **Задача 6** | Конзолно меню (`switch-case`) за пресмятане на лице на геометрична фигура по избор: триъгълник, квадрат, правоъгълник или кръг. |
