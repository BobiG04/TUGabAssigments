# 📐 Курсова работа: Калкулатор за лица на трапеци

## 🎯 Цел на проекта
Това е финалната курсова работа за първи курс в Технически университет – Габрово. Програмата представлява конзолно приложение, което пресмята лицата на до 50 трапеца, използвайки различни математически формули в зависимост от въведените параметри. След изчисляването, програмата запазва резултатите в масив и ги сортира във възходящ ред.

## ⚙️ Основна функционалност
* **Динамичен избор на формула:** Програмата автоматично решава коя формула да използва въз основа на това кои параметри (страни, височина, радиус) са въведени със стойност, различна от нула.
* **Математически формули:**
  1. По две основи и височина: $S = 0.5 \cdot (a + b) \cdot h$
  2. По четири страни: $S = \frac{a + c}{4(a - c)} \sqrt{(a + b - c + d)(a - b - c + d)(a + b - c - d)(-a + b + c + d)}$
  3. По радиус на описана окръжност: $S = 8 \cdot r^2$
* **Сортиране (Bubble Sort):** Използван е алгоритъм за сортиране на масиви за подреждане на пресметнатите лица във възходящ ред.
* **Модулност:** Кодът е разделен на отделни функции (`EnterBrTraps`, `EnterAndCalSums`, `SumTrap`, `SortSes`) за по-добра четимост и преизползваемост.
