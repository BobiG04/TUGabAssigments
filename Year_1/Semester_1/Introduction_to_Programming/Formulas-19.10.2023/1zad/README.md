# 📚 Пресмятане на математически формули

Тази директория съдържа 14 вариации на задачи, решавани в часовете по "Увод в програмирането" (19.10.2023 г.). Целта на заданията е упражняване на работата с променливи с плаваща запетая (`float`), конзолен вход и изход, както и използване на стандартните математически функции в C/C++ (`sqrt`, `pow`, `abs`).

## 🗂️ Списък с формулите

В таблицата по-долу са представени математическите изрази, които всяка програма пресмята.

| Вариация              | Математическа формула / Израз | Бележки                                                                                                                                                                 |
| :---------------------------- | :----------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Вариация 1**  | y = √(a² + 2ab + b²)                                | Базово използване на `sqrt` и `pow`.                                                                                                                    |
| **Вариация 2**  | y = (2a + b³) / √8                                   | Деление с корен в знаменателя.                                                                                                                        |
| **Вариация 3**  | y = ((5 - x³) / √3) + (2x / 8)                       | Събиране на дробни изрази.                                                                                                                               |
| **Вариация 4**  | y = ((a + b)² - 4ab) / √(3\|a\|)                     | Използване на модул (`abs`) в корена.                                                                                                                |
| **Вариация 5**  | y =\|√(2 + x⁵) / (x⁴ + 5x)\|                        | Влагане на функции вътре в модул.                                                                                                                   |
| **Вариация 6**  | y = ((5 - x⁵) / 3) - x + 1                            | Линейни операции с дроби и степени.                                                                                                               |
| **Вариация 7**  | y = ((x + a)⁵ +\|x + 2a\|) / (5a + √(a⁵))           | Комбиниране на степени, модул и корен.                                                                                                          |

## 🚀 Как се компилира и стартира

Всяка вариация е напълно самостоятелна програма.

1. Отвори терминала в директорията на задачата.
2. Компилирай желания файл (напр. за Вариация 1): `g++ variant_01.cpp -o variant_01`
3. Стартирай компилирания файл: `./variant_01`

*(Забележка: Програмите използват специфичния за Windows/MSVC `scanf_s`. За компилиране с GCC компилатор под Linux/macOS, заменете `scanf_s` със стандартния `scanf`.)*
