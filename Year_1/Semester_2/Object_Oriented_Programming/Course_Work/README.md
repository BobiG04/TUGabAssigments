# 🏨 Курсова работа: Система за управление на хотели

Тази директория съдържа финалната курсова работа по програмиране за първи курс (Катедра "Компютърни системи и технологии", ТУ-Габрово). Проектът представлява интерактивно конзолно приложение за съхранение и търсене на информация за хотели.

## ⚙️ Описание на проекта

Приложението позволява въвеждане и извеждане на данни за хотели (име, адрес, GPS координати), както и управление на събития към тях.

* **Основни функционалности:**
  * Интерактивно конзолно меню с команди (`Input`, `Output`, `Event Input`, `Event Output`, `Check Hotel`, `End`).
  * Търсене на хотел по име чрез статичен метод.
  * Работа с масиви от обекти (`Hotel[]`).
* **Обектно-ориентиран дизайн:** Проектът е разделен на главен клас `Program` и модел `Hotel`, демонстрирайки капсулация (properties) и работа с методи на екземплярите.
