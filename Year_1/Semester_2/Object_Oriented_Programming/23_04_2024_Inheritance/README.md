# 🗓️ Упражнение: 23.04.2024 - Наследяване (Inheritance)

Тази директория съдържа решения на задачите от Лабораторно упражнение № 9. Фокусът е върху един от основните стълбове на Обектно-ориентираното програмиране – наследяването (създаване на йерархии от класове, преизползване на код и извикване на конструктори на базовия клас чрез ключовата дума `base`). Всички класове са изнесени в отделни файлове за по-добра архитектура.

## 🗂️ Списък със задачите

| Проект       | Описание по условие                                                                                                                                                                                                                                                                                                                                                                                           |
| :----------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. _1zad** | **Геометрична йерархия:** Изграждане на верига от класове: `Dot` (Точка) -> `Circle` (Окръжност) -> `Cylinder` (Цилиндър) и `Cone` (Конус). Демонстрира наследяване на координати и радиус, както и надграждане с височина за пресмятане на обем. |
| **2. _2zad** | **Административна йерархия:** Базов клас `Person` (с име, ЕГН, номер на лична карта) и производен клас `Student` (добавящ специалност, група и факултетен номер). Демонстрира инициализация чрез `base()` конструктори.                                      |
