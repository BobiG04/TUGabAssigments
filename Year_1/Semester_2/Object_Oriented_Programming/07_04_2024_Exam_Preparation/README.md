# 🗓️ Упражнение: 02.04.2024 - Подготовка за изпит (ООП)

Тази директория съдържа комплексни задачи за подготовка за практическо изпитване по Обектно-ориентирано програмиране. Проектите обединяват знанията за класове, конструктори, свойства, методи и агрегация (обекти в обекти).

## 🗂️ Списък със задачите

| Проект       | Описание по условие                                                                                                                                                                                                                                                                                                                                                        |
| :----------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1. _1zad** | **Среден разход на гориво:** Клас `Car`, който следи модел, гориво и разход. Програмата приема команди за шофиране (`Карай <модел> <км>`), изчислява дали горивото ще стигне и обновява изминатото разстояние.         |
| **2. _2zad** | **Каталог за превозни средства:** Клас `Catalogue` за съхранение на коли и камиони. Приема неопределен брой превозни средства, отпечатва ги и накрая пресмята средните конски сили по категории.                                 |
| **3. _3zad** | **Мобилен телефон:** Агрегация на класове. Клас `Phone`, който съдържа в себе си обекти от тип `Battery` и `Display`. Включва статичен метод за история на обажданията и свойство за разпознаване на стационарни номера. |
